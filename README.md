# scrollText
===
This is a plugin about  `scrollText`
插件支持amd和cmd的加载方式，相当于一个小模块
###参数 param
####1.content
  	提供了两种填写list表单的方法<br/>
  	一种是直接填写content list<br/>
  	第二种是通过ajax获取后台返回的list<br/>
####2.speed
  	设置滚动的速度
####3.ajaxUrl
  	通过ajax获取list
####4.direction
  	这只滚动的方向
#####5.待续...
  	提供不同类型的滚动
